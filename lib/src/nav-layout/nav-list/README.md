## nav-list



Example for Nav List Layout:

```html
<nav-list sidenavTitle="title" logo="logo" icon="icon" opened="true" mode="side" sidenavWidth="350px" color="color" navigationRoute="/">
  <div sidenav-toolbar-content>
    ... left toolbar content
  </div>
  <md-nav-list sidenav-content>
    ... sidenav content
  </md-nav-list>
  <nav-toolbar-content>
    ... right toolbar content
  </nav-toolbar-content>
  ... main content
  <nav-footer-inner>
    ... sub footer content
  </nav-footer-inner>
  <nav-footer>
    ... main footer content
  </nav-footer>
</nav-list>
```
